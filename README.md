# javascript-tetris

JavaScriptとHTML5 Canvasで作成したテトリスゲームです。

## 機能

- テトロミノの移動
- テトロミノの回転
- ゴーストピース
- ハードドロップ
- NEXT表示
- スコア表示
- レベルシステム
- ライン消去

## 操作方法

| キー | 操作 |
|------|------|
| ← | 左移動 |
| → | 右移動 |
| ↓ | ソフトドロップ |
| Space | 回転 |
| Enter | ハードドロップ |
